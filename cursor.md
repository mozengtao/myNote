[agent modes](https://cursor.com/cn/docs/agent/modes)  
- Agent
   复杂功能、重构
   自主探索、多文件编辑
- Ask
   学习、规划、提问
   只读探索，无自动修改
- Plan
   需要规划的复杂功能
   在执行前创建详细计划，并提出澄清性问题
- Debug
   棘手 Bug、回归问题
   生成假设、日志埋点、运行时分析
