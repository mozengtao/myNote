### 🌐 云端开发与实验平台访问指南

* [**GitHub Codespaces**](https://github.com/features/codespaces) - 全能型在线 IDE，与 GitHub 仓库深度集成。
* [**Project IDX**](https://idx.google.com) - Google 推出的内置 AI 辅助的全栈开发环境。
* [**Killercoda**](https://killercoda.com) - 交互式 Linux 与 Kubernetes 学习实验平台。
* [**Play with Docker**](https://labs.play-with-docker.com) - 官方提供的 Docker 多节点集群练习场。
* [**Google Cloud Shell**](https://shell.cloud.google.com) - 带有 5GB 永久存储的云端 Linux 终端。
* [**Google Colab**](https://colab.research.google.com) - 提供免费 GPU 算力的交互式笔记本环境。
* [**Replit**](https://replit.com) - 支持多语言快速协作与代码运行的轻量级平台。
* [**DistroSea**](https://distrosea.com/) - 支持在线体验多种linux发行版的轻量级平台。
---

### 💡 如何选择最适合你的工具？

1. **如果你要写代码并保存项目：** 首选 **GitHub Codespaces**。它的环境配置最专业，且能与你的代码库无缝同步。
2. **如果你要练习 Linux 系统操作或 K8s：** 首选 **Killercoda**。它解决了 DistroSea 不能联网和无法保存进度的痛点。
3. **如果你需要一个随身携带的“保险箱”终端：** 首选 **Google Cloud Shell**。只要有浏览器，你的脚本和工具配置永远都在。
4. **如果你在跑深度学习：** 无需犹豫，直接使用 **Google Colab** 薅 GPU 羊毛。
