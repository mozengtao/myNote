# Advanced Prompting Playbook — Learning Mode

This playbook is designed for people who want to **learn deeply, quickly, and accurately** using ChatGPT.

Focus:
- Build real understanding
- Avoid shallow summaries
- Stress-test knowledge
- Retain long-term

Use this like a toolkit.

---

# 0. Core Philosophy

Do NOT ask:
> “Explain X”

Instead run this loop:

1. Build mental model  
2. Test understanding  
3. Apply  
4. Stress test  
5. Teach back  

ChatGPT works best when used as:
- Tutor
- Interviewer
- Reviewer
- Adversary
- Simulator

---

# 1. The Learning Loop (Primary Workflow)

Use this for any topic.

```markdown
You are my technical tutor.

Goal:
Help me deeply understand: [TOPIC]

Assume:
- I want real understanding, not summary
- I prefer precise explanations
- I want to be tested

Process:
1. Explain core idea simply but accurately
2. Show internal mechanics
3. Show common misconceptions
4. Ask me 3 probing questions
5. Give me a small exercise
6. Wait for my answer before continuing
```

This turns ChatGPT into an **interactive tutor**.

---

# 2. Mental Model Builder Prompt

Use when starting something new.

```markdown
Teach me the mental model of: [TOPIC]

Structure:
1. What problem does it solve?
2. Why was it invented?
3. What breaks without it?
4. Core abstraction
5. How it works internally
6. Real-world analogy (only if useful)
7. Where people get confused
8. One diagram in text form
```

---

# 3. Deep Dive Prompt (Expert Mode)

Use when you already know basics.

```markdown
Assume I know the basics of [TOPIC].

Go deep into:
- Internal mechanics
- Tradeoffs
- Edge cases
- Failure modes
- Performance implications

Avoid:
- Beginner explanations
- History lessons
- Fluff
```

---

# 4. Socratic Tutor Mode

Best for retention.

```markdown
Act as a strict tutor.

Goal:
Test my understanding of [TOPIC].

Rules:
- Ask one question at a time
- Increase difficulty gradually
- Force me to reason
- Do not reveal answers immediately
- Challenge weak answers
```

---

# 5. Feynman Technique Prompt

Use to solidify understanding.

```markdown
I will explain: [TOPIC]

Your job:
1. Identify gaps
2. Identify incorrect assumptions
3. Ask clarifying questions
4. Force me to refine explanation
5. Only accept when explanation is clear and correct
```

---

# 6. Confusion Killer Prompt

When something feels fuzzy.

```markdown
I am confused about: [TOPIC]

Help me by:
1. Identifying likely confusion points
2. Showing minimal working example
3. Showing wrong mental model
4. Showing correct mental model
5. Giving a tiny test question
```

---

# 7. Compare & Contrast Prompt

Great for deeper clarity.

```markdown
Compare:
[A] vs [B]

Focus on:
- Why both exist
- Core differences
- Tradeoffs
- When each fails
- When to choose each

Output:
Table + explanation
```

---

# 8. “Make Me Dangerous” Prompt

Use when you want practical skill.

```markdown
Goal:
Make me dangerous in [TOPIC] quickly.

Do:
1. Core concepts
2. 20% that gives 80% power
3. Real-world usage
4. Common mistakes
5. Mini project idea
```

---

# 9. Simulation Prompt

For applied learning.

```markdown
Simulate a real-world scenario where I must use [TOPIC].

Steps:
1. Present problem
2. Let me attempt solution
3. Critique my approach
4. Show expert solution
5. Compare differences
```

---

# 10. Adversarial Learning Prompt

For mastery.

```markdown
Your role:
Try to expose weaknesses in my understanding of [TOPIC].

Do:
- Ask tricky questions
- Present edge cases
- Challenge assumptions
- Force precise thinking
```

---

# 11. Layered Learning Prompt

Use for complex topics.

```markdown
Teach [TOPIC] in layers:

Layer 1: Intuition  
Layer 2: Mechanics  
Layer 3: Implementation  
Layer 4: Edge cases  
Layer 5: Expert insights  

Pause after each layer.
```

---

# 12. Memory Reinforcement Prompt

For retention.

```markdown
Help me retain [TOPIC].

1. Summarize key ideas
2. Create 5 recall questions
3. Create 3 tricky questions
4. Create spaced repetition prompts
```

---

# 13. Project-Based Learning Prompt

Best for real skill.

```markdown
Design a small project to learn: [TOPIC]

Include:
- Requirements
- Constraints
- Steps
- Hints
- Review criteria
```

---

# 14. Daily Learning Session Template

Use daily.

```markdown
Today I’m learning: [TOPIC]

Time available: [X minutes]

Plan session:
1. Core concept
2. Quick exercise
3. Test understanding
4. Review mistakes
```

---

# 15. Anti-Hallucination Prompt

When accuracy matters.

```markdown
If unsure, say so.

Separate:
- Known facts
- Assumptions
- Simplifications

Avoid guessing.
```

---

# 16. Meta-Learning Prompt

Improve how you learn.

```markdown
Analyze how I am learning [TOPIC].

Identify:
- Weak areas
- Knowledge gaps
- Better learning path
- Practice ideas
```

### Rule 2

### Rule 3

### Rule 4

### Rule 5
